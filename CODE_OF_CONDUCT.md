# Code of Conduct

## Short version

Be respectful. Don't be a jerk. We're all here to build good software.

## Longer version

We want vibespace to be a welcoming project for everyone regardless of experience level, background, or identity.

**Expected behavior:**

- Be constructive in discussions and code reviews
- Accept feedback gracefully
- Focus on what's best for the project
- Be patient with newcomers

**Unacceptable behavior:**

- Harassment, insults, or personal attacks
- Trolling or deliberately inflammatory comments
- Publishing others' private information
- Any conduct that would be inappropriate in a professional setting
